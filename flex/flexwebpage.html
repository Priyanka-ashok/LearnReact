Basic Concept of Flex

  • Flex is based on flex-flow directions
  • Default value : flex-direction: row;

Flex Container

  • Display: flex ; This defines the Flex-container
  • Flex-Properties : justify-content, align-items, align-self, flex-grow, order, flex-basis, flex-shrink

Flex Flow Shorthand

  • It takes the three flex properties
  • flex grow, flex-shrink,flex-basis
  • Example : 1 1 150px;
